Getting Started

To get your MICROWIND2 program started, use the following procedure:

  • Insert the MICROWIND2 CD-Rom into drive

  • Under Windows 95/NT, click on Start -> Execute

  • Type D:install and press ¿

  • Double click on the Microwind2 icon to start the software

The software runs on Windows 95, 98 and NT operating systems.

The command line may include two parameters: The First parameter is the default mask file loaded at initialization. The Second parameter is the design rule file loaded at initialization. Example : The command " microwind2 fadd cmos018.rul " executes MICROWIND2 with a default mask file " fadd.MSK " and the rule file " cmos018.RUL ". You may program the Microwind2 icon by a click with the right button, then "properties".
