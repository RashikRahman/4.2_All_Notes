Import CIF (File Menu)

The command File -> Import CIF (Professional version) or File -> Open (with type "CIF file") may be used to import a CIF layout description generated by an other layout tool such as CADENCE, MENTOR, ZUKEN, etc..

 

The rule file should contain the correspondence between Microwind layers and CIF layers. In the .RUL file, the CIF description syntax is as follows: "CIF <Microwind layer> <import CIF layer> <overetch>". In the example below, the Microwind layer "metal" is described as "mtl1" in the CIF file. All metal boxes are resized with a negative 0.01 µm when importing from CIF, and a positive 0.01µm when generating the CIF file to export the layout. The overetch is mainly used in the case of contacts, as the 2 lambda basic rule for contact width has been kept standard for all technologies.

cif metal mtl1 0.01
